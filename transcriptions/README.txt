Folder na pliki transkrypcji

W tym folderze zostaną zapisane pliki transkrypcji w formacie Markdown (.md).
Każdy plik transkrypcji zawiera:
- Nagłówek z informacjami o pliku źródłowym
- Datę i czas wykonania transkrypcji
- Pełny tekst transkrypcji
- Podział na segmenty czasowe (znaczniki czasu)

Pliki Markdown można otworzyć w dowolnym edytorze tekstu lub w dedykowanych aplikacjach jak:
- Obsidian
- Visual Studio Code
- Typora
- Notion

Możesz edytować pliki transkrypcji, aby poprawić ewentualne błędy lub dodać własne adnotacje.
